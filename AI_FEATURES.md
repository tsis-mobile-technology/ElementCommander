## 기술 아키텍처

### AI 클라이언트 구조
```
┌─────────────────────────────────┐
│   UI Layer (Alt+*, Ctrl+*)      │
├─────────────────────────────────┤
│   Command Handler (app.rs)      │
├─────────────────────────────────┤
│   AI Module (ai/mod.rs)         │
│   - 기능별 함수들               │
├─────────────────────────────────┤
│   HTTP Client (ai/client.rs)    │
│   - OpenAI 호환 API             │
├─────────────────────────────────┤
│   LLM Server (llama.cpp)        │
│   (localhost:8080)              │
└─────────────────────────────────┘
```

### 상태 관리
- **ai/state.rs**: 배치 작업, 진행도, 에러 처리
- **config.rs**: notes.json 읽기/쓰기, 설정 관리

### 에러 처리
- LLM 서버 unavailable → Graceful degradation
- Timeout → 사용자 알림 및 재시도 옵션
- API 오류 → 상세 로그 및 디버깅 정보
